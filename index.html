Portfolio

Stake - 2020

Personally built a web app that allows users to manage and keep track of volunteer opportunities in Chicago. Stake allows users to create an account, look through available volunteer opportunities in their neighborhood, and ultimately choose opportunities that they wish to partake in. Users also have the ability to add their own volunteer opportunities to the database which is kept and managed using PostgreSQL. Built with Ruby on Rails along with Vue.js with Bootstrap in the front-end. Also integrated Javascript third party libraries such as Moment.js, Vue2Filters, and Mapbox-gl; as well as Cloudinary to allow users to add/update images to their profiles. Web requests were completed using the Axios NPM.

This project is a milestone for me as it is the first application that I was able to build on my own while attending a coding bootcamp. Stake is the culmination of what I learned at my time at the bootcamp as well as me as an individual. With this project, I was able to create something that is meant to build and empower commmunities. It is my desire that I will able to do so throughout the rest of my career.

Languages: Ruby, Javascript, HTML, and CSS

Frameworks: Rails, Vue.js, and Bootstrap

RatemyInstructor - 2021

RateMyInstructor is a web app that allows individuals to view and create reviews for their professors, based on the RateMyProfessor website. RateMyInstuctor allows users to assign ratings for professors and write detailed reviews. Users are able to see a professorâ€™s overall score based on the average of their ratings as well. Node.js and Express.js were used in the back-end, and Vue.js with Bootstrap in the front-end. The database is kept and managed using PostgreSQL. VanillaJS Video Player - 2021

Predominantly used Vanilla Javascript to create a single-page web app that features a custom video player. Users can control videos with traditional video player controls (i.e. play, pause, volume). Speech Recognition was implemented to allow users to control the video based on voice commands. Color filters can also be applied to videos, which is done using the canvas element in HTML. Users are able to upload video files that are playable using the video player. Cloudinary is used to host uploaded files with API credentials. HTML and CSS were used to style to web app.

The VanillaJs Video Player was built with Alex Racinski and Erik Alvar, who are also Actualize alumni. Throughout the construction of the app we used paired programming, and strictly relied on pull requests on github for version management.

Languages: Javascript, HTML, and CSS
